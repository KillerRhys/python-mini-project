Mega Millions®

Download Current and Past Winning Numbers

The comma separated values file you can download here contains current and past winning numbers from all Mega Millions drawings held since Texas began participating in game drawings on December 5, 2003.

The information in the file is organized in the following manner, one row for each drawing:

Game Name, Month, Day, Year, Num1, Num2, Num3, Num4, Num5, Mega Ball, Megaplier
